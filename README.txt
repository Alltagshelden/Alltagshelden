🎮 Alltagshelden – Remote Support
Preview 0.3 Light

Willkommen im kleinen Büro-Legenden-Game! Wir (Manuel, Marek, Dirk) sind Servicetechniker im Remote Support Office
und haben unseren Alltag mit AutoStore, Managern & ITlern in Pixel-Form gegossen.

🚀 Start
1) ZIP entpacken
2) index.html im Browser öffnen (Chrome oder Edge empfohlen)
3) Losspielen. Keine Installation, keine Adminrechte nötig.

🎮 Steuerung
- Bewegen: A/D oder ←/→
- Springen: W / ↑ / Leertaste
- Schießen: F / Strg
- Waffen wechseln: 1–5
- Spezialfähigkeit: Q (Platzhalter)
- Pause/Legende: H oder ESC

📝 Hinweis
- Highscores werden lokal im Browser gespeichert (Top 10).
- Sprüche & Audio sind vorerst Platzhalter.
- Changelog ist in dieser Preview in der README – bald direkt im Spiel unter „Was ist neu?“ abrufbar.
